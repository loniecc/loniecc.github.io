Flink 本地环境搭建

Idea 直接运行式

  1. 下载Flink
  2. 创建项目
  3. 用Idea 打开创建好的项目
  4. 在Idea 打开配置File-Project Structure-Libraries-±–java,添加第一步解压出来的Flink包里的lib 和 opt 包
  5. 随便写点啥，运行吧，然后就能在控制台打印出来了
  6. 特别注意
     1. 不要引入Spring Boot，不要仗着Idea是高级版就无脑创建Spring Boot项目
 2. 没有主包的，create 一个，然后mark一下
